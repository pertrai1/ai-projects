## ADDED Requirements

### Requirement: Analysis
The project SHALL analyze the evaluation results.

#### Scenario:
- **WHEN** the analysis is run
- **THEN** the accuracy for each prompt type and task category is calculated.

### Requirement: Console Output
The project SHALL print the analysis results to the console.

#### Scenario:
- **WHEN** the analysis is run
- **THEN** the analysis results are printed to the console in a human-readable format.
