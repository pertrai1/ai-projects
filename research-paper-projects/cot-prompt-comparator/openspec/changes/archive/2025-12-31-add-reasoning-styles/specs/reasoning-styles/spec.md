## ADDED Requirements

### Requirement: Concise Chain-of-Thought Prompt
The project SHALL have a concise chain-of-thought prompt that takes a question and returns a formatted prompt.

#### Scenario:
- **WHEN** the `conciseChainOfThoughtPrompt` function is called with a question
- **THEN** it returns a string with the question formatted as a concise chain-of-thought prompt.

### Requirement: Verbose Chain-of-Thought Prompt
The project SHALL have a verbose chain-of-thought prompt that takes a question and returns a formatted prompt.

#### Scenario:
- **WHEN** the `verboseChainOfThoughtPrompt` function is called with a question
- **THEN** it returns a string with the question formatted as a verbose chain-of-thought prompt.
