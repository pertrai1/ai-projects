## ADDED Requirements

### Requirement: Standard Prompt
The project SHALL have a standard prompt that takes a question and returns a formatted prompt.

#### Scenario:
- **WHEN** the `standardPrompt` function is called with a question
- **THEN** it returns a string with the question formatted as a standard prompt.

### Requirement: Chain-of-Thought Prompt
The project SHALL have a chain-of-thought prompt that takes a question and returns a formatted prompt.

#### Scenario:
- **WHEN** the `chainOfThoughtPrompt` function is called with a question
- **THEN** it returns a string with the question formatted as a chain-of-thought prompt.
